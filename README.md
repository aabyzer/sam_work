# Коллекция рецептов

## 📋 Возможности

- **Просмотр всех рецептов** на главной странице.
- **Добавление нового рецепта** через форму (POST-запрос).
- **Просмотр деталей** конкретного рецепта по ID.
- **Редактирование** существующего рецепта.
- **Удаление** рецепта.
- **Поиск** рецептов по названию
- **Сортировка** по категориям:
  - `/sort/soup` – только супы
  - `/sort/salad` – только салаты
  - `/sort/desserts` – только десерты
- Все изменения сохраняются в базе данных SQLite.

## 🚀 Установка и запуск

### 1. Клонирование репозитория

Если проект ещё не загружен, выполните:

```bash
git clone https://github.com/ваш-логин/sam_work.git
cd sam_work
python -m venv venv
venv\Scripts\activate
```

### 2. Установка зависимостей
```bash
pip install flask

pip install pytest
```

### 3. Запуск сервера
```bash
python app.py
```

Приложение будет доступно по адресу: http://127.0.0.1:5000

Чтобы остановить сервер – нажмите Ctrl+C.
